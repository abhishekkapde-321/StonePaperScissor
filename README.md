# StonePaperScissor
🪨📄✂️ Stone Paper Scissor Game

A simple and interactive Stone Paper Scissor (Rock Paper Scissors) game built using HTML, CSS, and JavaScript.
Play against the computer with a clean UI and real-time score updates.

📌 Project Overview

This project is a browser-based implementation of the classic hand game:

🪨 Stone beats Scissor

✂️ Scissor beats Paper

📄 Paper beats Stone

The game randomly generates the computer's move and compares it with the player's choice to determine the winner.

🎮 Features

🎨 Clean and responsive UI

🤖 Computer-generated random moves

🧮 Real-time score tracking

🏆 Win/Lose/Draw result display

🔄 Reset button to restart the game

📱 Mobile-friendly design

🛠️ Technologies Used

HTML – Structure of the game

CSS – Styling and layout

JavaScript – Game logic and interactivity

🧠 How the Game Works

Player selects:

Stone

Paper

Scissor

Computer randomly selects one option.

Game logic compares both selections:

If both are same → Draw

If player wins → Player score +1

If computer wins → Computer score +1

Result is displayed instantly on the screen.
